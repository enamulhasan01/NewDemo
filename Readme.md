# Just for Prac

This project is a simple and easy-to-understand application created to learn and practice programming concepts.  
It focuses on writing clean code, understanding basic logic, and improving problem-solving skills.

The project is suitable for beginners and anyone who wants a clear example of how the core features work.  
More features and improvements will be added over time as the project grows.

Feel free to explore, use, and modify the code for learning purposes.
